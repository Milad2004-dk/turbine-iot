# Wind Turbine IOT API

En REST API til overvågning af vindmøller med temperatur-alarm system.

## Funktioner
- Opret og administrer vindmøller
- Modtag sensor data fra IoT enheder
- Automatisk alarm hvis temperatur overstiger 75°C
- Alarm data gemmes i MySQL database
- Email notifikation ved alarm

## Teknologier
- Python / Flask
- MySQL (Azure Database)
- Swagger / Flasgger
- Gmail SMTP
